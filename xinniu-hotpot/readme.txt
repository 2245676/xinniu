Premium WordPress theme for Xinniu Hotpot, a high-end Chinese hotpot restaurant website for Fukuoka.

== Current Modules ==

* Theme bootstrap and modular includes.
* Gutenberg-ready theme.json palette and layout settings.
* Responsive base templates.
* Custom post types for dishes, news, FAQ, stores, home sections, malatang items, and promotions.
* Taxonomies for dish categories, tags, spice levels, allergens, news, FAQ, and malatang categories.
* Theme settings under Appearance > Xinniu Hotpot.
* Native meta boxes for menu, store, malatang, CTA, and SEO fields.
* SEO meta, canonical, robots, Open Graph, Twitter Card.
* JSON-LD Restaurant, WebSite, Breadcrumb, Article, Menu, MenuItem, and FAQ output.
* llms.txt endpoint.
* Malatang SEO page template.
* FAQ page template and FAQ category archives.
* Gutenberg block patterns for reservation and malatang content.
* Mobile-first homepage, menu, news, FAQ, footer, and quick action UI.
* WooCommerce compatibility hooks and styles.

== Setup ==

1. Copy the xinniu-hotpot folder into wp-content/themes/.
2. Activate the theme in Appearance > Themes.
3. Visit Settings > Permalinks and save once after activation if rewrite rules need refreshing.
4. Configure Appearance > Xinniu Hotpot.
5. Create a page with the "Malatang SEO Page" template and assign the /malatang/ slug.
6. Create a page with the "FAQ" template and assign the /faq/ slug.

== Notes ==

This theme intentionally avoids hard-coding restaurant operating content. Use theme settings, pages, CPT entries, taxonomies, and meta fields for editable content.
